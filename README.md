# Start Here

Hey, welcome. This is the Summer 2026 Coding Club. It's a place to learn to code, build things that are actually yours, and get help when you need it.

No schedule, no deadlines. Work whenever you want, at whatever level you're at, and reach out anytime you want feedback or a nudge in the right direction. That's what we're here for.

## What it's about

Most of you said you're into AI / machine learning and data science, so that's where most of our resources point. Beginner or already shipping projects, either way there's a place to start.

## Where to start
- [Getting set up](https://github.com/Summer-2026-Coding-Club/intro-to-git/blob/main/setup-git.md) - get Python running and your tools ready
- [Intro to Git](https://github.com/Summer-2026-Coding-Club/intro-to-git) - how we share and work on code together
- [Coding with LLMs](https://github.com/Summer-2026-Coding-Club/intro-to-coding-with-llms) - using AI to learn and build faster

More gets added over time. This is just the start.

## How it works

- **Async and self-paced.** No lockstep, go at your own speed.
- **Build real things.** The goal is work you can show an employer or put on a resume, not just exercises.
- **Ask for help.** That's what we're here for, and if it's something we don't know either, we'll figure it out with you.
- **Meet up if you want.** If there's interest, we'll set up a call. Totally optional.
